Cameron> Is there an easy formula for getting SID FREQ HI/LO values
Cameron> from an arbitrary frequency value in Hz?

Yes, there is:

	frequency = value / 16777216 * Phi2

or

	value = frequency * 16777216 / Phi2

where

	Phi2 = 14318181 / 14 Hz  (NTSC)
or
	Phi2 = 17734472 / 18 Hz  (PAL).


The SID has a 24-bit waveform accumulator, and it adds the frequency
value to it on every clock cycle.  The most significant 8 bits of the
waveform accumulator (after the waveform function has been applied to
it) are used for output.
